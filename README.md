# HealthArc(A Healthcare Application)

HealthArc is healthcare application where patient can find the doctors and request for appointments.
##### Tech Stack 
       Spring Boot and Spring Security
       ReactJS and Tailwind CSS
       MongoDB
       Cloudinary 
       Docker and Git
